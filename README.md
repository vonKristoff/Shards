iShards
======

A jQuery plugin that builds a CSS multi-layered gradient background on any element - resembling neon shards!

```html
<h2>Example of code</h2>

<pre>
    <div class="container">
        <div class="block two first">
            <h2>Your title</h2>
            <div class="wrap">
            //Your content
            </div>
        </div>
    </div>
</pre>
```
